YouTube Playlist Downloader
===========================

A simple desktop application to download YouTube playlists.

Features:
- Download entire playlists or individual videos
- Audio-only (MP3) or video downloads
- Select specific videos to download
- Real-time progress tracking
- Custom download locations

System Requirements:
- Windows 7 or newer
- 50MB free disk space
- Internet connection

Installation:
1. Run the setup executable
2. Follow the installation wizard
3. Launch from Start Menu or Desktop
